 Beschreibung

KaM Remake (wir haben noch keinen besseren Titel) ist ein Versuch das originale "Knights and Merchants: The Shattered Kingdom" wiederzubeleben. Unsere Ziele sind es, das Spiel auf heutigen Computern und Betriebssystemen zum Laufen zu bringen, ebenso die vielen Fehler im Spiel zu beheben. Um dies zu machen haben wir angefangen unseren eigenen Code für die Engine zu schreiben, greifen jedoch auf viele originale Dateien zurück. Dies ist unsere öffentlich begrenzte Technologievorschau (Demo) mit der wirtschaftlichen und der kriegerischen Seite des Spiels; bezeichnet als "Modifikation" oder "Neuauflage". Unsere Absicht ist es, das Potenzial zu demonstrieren und Feedback über die Funktionalität zu bekommen. Alle von KaM entlehnten "Eigentümer" (Grafiken, Musik, Sounds, etc.) sind nur da, um die Absichten zu testen. Die finale Version wird keine, mit Copyright versehene, Dateien enthalten.
Wir gehen davon aus, dass sie das originale KaM besitzen, andernfalls sollten sie es sich kaufen. Es ist in mehreren Onlineshops verfügbar.
Die Kam Remake Exe-Datei ist Freeware und wurde von Liebhabern des Spiels erstellt. Die Anwendung benutzt:
Delphi 6, Delphi 7, FastMM4, Lazarus, OpenGL, OpenAL, zLib, PNGImage.
 

 Systemanforderungen

KaM Remake benötigt:
    - Microsoft Windows 98, XP, Vista oder Sieben;
    - 500MHz oder einen schnelleren Prozessor;
    - 256MB oder mehr RAM;
    - 3D Grafikkarte OpenGL 1.4 fähig;
    - Soundkarte.
 

 Installation

1. Entpackt alle Dateien aus dem Paket in einen neuen Ordner auf eurem PC.
2. Das Paket beinhaltet die OpenAL Audiotreiber (oalinst.exe), welche ihr installieren solltet um das KaM Remake zu spielen. Die neuste Version kann unter der offiziellen OpenAL-Seite gefunden werden: http://connect.creativelabs.com/openal/default.aspx
3. Das Remake benutzt die OpenGL Rendering Engine, ihr müsst also die OpenGL-Treiber installiert haben. Besucht die Seite des Herstellers eurer Grafikkarte um die neusten Treiber zu bekommen.
4. Startet die Modifikation - KaM_Remake.exe.
 

 Gameplay

The following shortcuts are available in the game:
F8 erhöht Spielgeschwindigkeit um x10
F11 zeigt Menü und Debugging Panel
1-4 öffnet jeweilige Auswahl im Spielmenü
P pausiert das Spiel
Esc schließt offene Nachricht oder Auswahl
Enf löscht offene Nachricht
Leertaste bewegt das Sichtfeld zum Ort der letzten Nachricht
←↑→↓ Pfeiltasten bewegen das Sichtfeld
Drehen des Mausrads zoomt heran und heraus
← Backspace setzt den Zoom auf 100% zurück
Klick auf Mausrad platziert Späher auf Position des Mauszeigers um Map zu erkunden

Zusätzliche Karten installieren:
1. Erstellt einen Unterordner im Verzeichnis 'Maps' und bennent ihn zum Beispiel "Meine neue Karte".
2. Fügt die .dat- und .map-Datei ein und benennt sie entsprechened "Meine neue Karte.dat" und "Meine neue Karte.map". Die Dateinamen sollten exakt dem Ordnernamen gleichen.
3. Ihr könnt optional auch eine Datei namens "Meine neue Karte.txt" anlegen, welche die Missionsbeschreibung
und andere Informationen enthalten wird. Benutzt das selbe Format wie in den Missionen die das Remake
anbietet. Keine offizielle Formatierung wird veröffentlicht, da sich das Format wahrscheinlich noch
ändern wird.

Benutzerdefinierte Musik hinzufügen:
In das Verzeichnis 'Music' könnt ihr MP3-Dateien einfügen, sie werden automatisch in die Playlist geladen.
 

 F.A.Q.

Abstürze ohne Benachrichtigung:
KaM Remake stürtzt beim Laden oder beim Wechseln der Musik ab, wenn es irgendwelche Probleme mit dem Windows Sound System gibt. Den Ordner 'Music' zu entfernen, sollte eine Lösung sein.

Niedrige Framerate:
Falls das Programm eine niedrige Framerate hat, könnte es sein, dass die OpenGL-Treiber nicht mehr aktuell sind. Ihr könnt die OpenGL-Version in der Ecke links oben im Hauptmenü sehen. Es sollte mindestens Version 1.4.x sein.

Fehler im Spiel:
Manchmal wenn etwas Unerwartetes passiert, bekommt ihr eine Fehlermeldung "An error has occoured during
gameplay". Please read the message details and send the crash report ZIP file from the Crash Reports folder by email either to Krom (kromster80@gmail.com oder lewinjh@gmail.com). Teilt
uns mit was ihr gemacht habt bevor die Fehlermeldung kam und schreibt möglicherweise andere Informationen
dazu, die uns von Nutzen sein können.

Assertion failed:
Auch erscheint manchmal eine "Assertion failed"-Nachricht. In diesem Fall solltet ihr KaM_Remake mit dem Task Manager (drückt Strg+Alt+Enf) beenden.
Dann geht in das KaM Remake-Verzeichnis und findet die Datei 'KaM.log'. Sie beinhaltet Details des Absturzes. Bitte sendet die Datei per E-Mail an Krom (kromster80@gmail.com) oder Lewin (lewinjh@gmail.com). Hängt bitte auch die Replay-Dateien an.

 Dedicated Server
Executables for the multiplayer dedicated server are included, available for Windows and Linux x86 and x86_64. The server listens for connections on TCP port 56789 and acts as a hub to trasfer the data between clients. The dedicated server does not understand gameplay, it simply moves packets around. The first client to join will be assigned hosting rights allowing them to setup the game. If that client disconnects, the next one in the list will be assigned hosting rights.
To use in a LAN game, start the dedicated server and ensure it is running properly (it should say "Listening on port 56789") Please ensure your firewall is allowing clients to connect to the server, by allowing TCP port 56789. You may then connect to it from the same computer and/or other computers. (for the same computer, connect to the address 127.0.0.1 or localhost)
To use on the internet, you must forward the port 56789 from your router to your PC running the dedicated server. (for more information on port forwarding visit http://portforward.com/) Tell your friends to connect to your external IP address, is displayed to you on the homepage of http://portforward.com/. If you are playing on the same computer that is running the dedicated server, or on the same LAN, connect to its IP address as you would for a LAN game.

Note: The dedicated server is not required to play multiplayer. Clicking "Create Server" from the network menu has the same effect, except you must participate in the game and the server cannot be left to maintain itself. The dedicated server is only necessary if you want to provide a server without playing. The main purpose of the dedicated server is to allow servers to be hosted 24/7, which players can join and participate in any time they like. If you are able to provide this service and tell other fans the address of your server we would be pleased. Future versions will include a "master server" which displays a listing of all running public servers and allow you to join them without having to type in IP addresses.

The Linux builds have been tested on Fedora and Ubuntu. Any queries or comments regarding the Linux builds should be sent to Lewin. (lewinjh@gmail.com)
